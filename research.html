科研经历 (Research Experience)

我的研究主要围绕可信学习型优化展开，关注如何将优化算法、 约束结构、反馈控制和对偶证书转化为可解释、可验证、可微分的学习型优化系统。

My research focuses on trustworthy learning-based optimization, with an emphasis on transforming optimization algorithms, constraint structures, feedback control, and dual certificates into interpretable, verifiable, and differentiable learning-based optimization systems.

代表性研究方向 (Selected Research Topics)

残差控制乘子学习 (Residual-Controlled Multiplier Learning)

研究随机约束优化中乘子记忆的稳定更新机制，通过 projected pressure 与 pressure-memory residual 的分离提升约束控制与乘子稳定性。

学习增强优化算法 (Learning-Augmented Optimization)

研究如何将传统优化算法展开为可训练的反馈控制系统，并引入约束残差、 对偶间隙和安全机制提升可解释性与可靠性。

综合能源系统优化调度 (Integrated Energy System Scheduling)

研究风、光、氢、储能和柴油/燃气机组协同运行下的多能系统优化调度问题。

论文与工作 (Publications and Manuscripts)

Residual-Controlled Multiplier Learning for Stochastic Constrained Decision-Making
Kang Liu, Jianchen Hu, Ziyu Qu.
arXiv:2606.07088, cs.LG, 2026.

This work develops a residual-controlled multiplier learning framework for stochastic constrained decision-making, improving feasibility control and multiplier stability under noisy mini-batch feedback.

Exact Dual Geometry of SOC-ICNN Value Functions
Kang Liu, Jianchen Hu, Wei Peng.
arXiv:2605.04722, cs.LG, 2026.

This paper characterizes SOC-ICNN value functions from the dual viewpoint, showing how subgradients, directional derivatives, and local Hessians can be recovered from optimal dual variables.

Budget-aware Auto Optimizer Configurator
Kang Liu, Wei Peng, Jianchen Hu.
arXiv:2605.04711, cs.AI, 2026.

This work proposes BAOC, a budget-aware optimizer configuration method that allocates optimizer mechanisms and precision across model blocks to reduce optimizer-state memory while preserving training quality.

SOC-ICNN: From Polyhedral to Conic Geometry for Learning Convex Surrogate Functions
Kang Liu, Jianchen Hu, Wei Peng.
arXiv:2604.22355, cs.LG, 2026.

This paper introduces SOC-ICNN, which extends ReLU-based ICNNs from polyhedral LP-type representations to conic SOCP-type representations with native smooth curvature.

OPBO: Order-Preserving Bayesian Optimization
Wei Peng, Jianchen Hu, Kang Liu, Qiaozhu Zhai.
arXiv:2512.18980, cs.LG, 2025.

This work proposes an order-preserving Bayesian optimization method that replaces precise value regression with ordinal surrogate learning for high-dimensional black-box optimization.

Biomed-DPT: Dual Modality Prompt Tuning for Biomedical Vision-Language Models
Wei Peng, Kang Liu, Jianchen Hu, Meng Zhang.
arXiv:2505.05189, cs.CV, 2025.

This paper proposes a dual-modality prompt tuning method for biomedical vision-language models, combining clinical text prompts with visual prompts to improve few-shot biomedical image classification.

Learning based convex approximation for constrained parametric optimization
Kang Liu, Wei Peng, Jianchen Hu.
arXiv:2505.04037, math.OC, 2025.

This work develops an ICNN-based self-supervised framework combined with augmented Lagrangian learning and constraint correction for solving constrained parametric optimization problems.

EEG-DCNet: A Fast and Accurate MI-EEG Dilated CNN Classification Method
Wei Peng, Kang Liu, Jiaxi Shi, Jianchen Hu.
arXiv:2411.17705, eess.SP, 2024.

This paper proposes EEG-DCNet, a multi-scale dilated CNN architecture for efficient and accurate motor-imagery EEG classification in brain-computer interface applications.
